幂药云对外接口文档

接入指南

如需接入幂药云相关服务请先联系线下商务人员。

在您详细了解过产品服务信息并确定对接范围后，平台将会为您分配相关产品服务的相关凭证，主要为AccessID、AccessSecret，请妥善保管。
目前该产品服务主要以API的方式为您提供服务，您可以选择各产品的文档，查看具体使用方法及参数。
同时，我们为您提供了技术支持，如有疑问，请联系技术人员，我们会在及时响应您的问题。

接口环境说明

测试环境地址：https://open-api.test.jk.com

接口调用说明

每个接口的调用路径：环境Url+接口请求Url

通过登录接口获取token
幂药云平台提供了api接口，供医院系统调用，完成前置审方。

  • 接口采用 Restful 风格的 WebApi 实现
  • 数据采用 Json 格式承载

登录接口（Login）Header定义：
lContent-Type:application/json
lAPP-KEY:1f6374cd6c364308b42bfbc993f795ac(此处为示例，联系幂药云业务人员获取)
lAPP-SECRET:96e5313d9c764a9fb3486543ed2d443b(此处为示例，联系幂药云业务人员获取)
其它接口（非登录接口）Header定义：
lContent-Type:application/json
lAPP-KEY:1f6374cd6c364308b42bfbc993f795ac(联系幂药云业务人员获取)
lAPP-SECRET:96e5313d9c764a9fb3486543ed2d443b(联系幂药云业务人员获取)
lAuthorization:Bear Token说明：Token为登录返回的Data数据，Token有时效性，过期后需重新调用Login获取
在调用业务接口之前，要调用登录接口获取Token，然后通过Http Header的Authorization字段认证身份，Authorization字段的内容格式为Bearer token（注意两者之间是一个空格）。Token有一定时间的有效期（默认120分钟），超过时间，Token失效，需重新调用登录接口获取Token

接口返回值说明

参数 返回值类型 说明
Code Int 接口返回编码
Message String 消息说明
Data Object 返回数据

接口参数规范

合理用等接口入参和出参使用小驼峰写法，如custDrugCode；
处方流转相关接口，入参和出参中业务字段采用大驼峰写法，如CustDrugCode；

处方审核

说明

基本信息

Path： /13

Method： GET

接口描述：

业务介绍

按照“构建1套系统、共享1套数据、通用1套规则、服从1套管理”的原则，为医院提供最全面的处方审核服务，配套产品主要为合理用药系统、药师前置审方系统和处方点评系统。线下实体医院、线上互联网医院、医联体及其他医疗机构均可接入处方审核中心。

应用场景

适用于能够开具处方的实体医院和互联网医院，接入幂药云后完成开处方闭环流程。如对应传输字段没有传入，处方将无法审核。
