# vue-cli-template

@jk998 前端团队 vue 模板库, 开箱即用

# 更新基础库

```sh
yarn add @jk998/library @jk998/element-ui-peace-dialog @jk998/element-ui-peace-table
```

## 框架

- `全量加载` @jk998/library
- `按需加载` quasar

## UI 库

- `全量加载` element-ui
- `全量加载` @jk998/element-ui-peace-dialog
- `全量加载` @jk998/element-ui-peace-table
- `按需加载` quasar
